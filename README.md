# Validador de Senhas

Projeto desenvolvido para fins acadêmicos utilizando a metodologia TDD.

# Tecnologias Utilizadas: 

- Python
- Pytest Framework

## Ciclo TDD Aplicado

O projeto seguiu rigorosamente o seguinte fluxo:

1. **Red**: Escrita de um teste que falha para cada requisito.
2. **Green**: Implementação do código mínimo para o teste passar.
3. **Refactor**: Limpeza do código e melhoria contínua da lógica sem quebrar os testes.

## Requisitos Validados

- Mínimo de 8 caracteres.
- Ao menos uma letra maiúscula.
- Ao menos uma letra minúscula.
- Ao menos um número.
- Ao menos um caractere especial.
- Proibição de espaços em branco.
